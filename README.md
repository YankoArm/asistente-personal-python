# Asistente Personal en Python

Aplicación de escritorio desarrollada en Python que permite gestionar tareas y notas mediante una interfaz gráfica (Tkinter) y un modo consola.

## Funcionalidades
- Gestión de tareas (crear, listar, marcar como hechas)
- Gestión de notas
- Persistencia en archivos JSON
- Interfaz gráfica con Tkinter
- Modo consola

## Estructura del proyecto
- core/: lógica principal del asistente
- features/: funcionalidades (tareas, notas)
- gui/: interfaz gráfica
- data/: almacenamiento de datos

## Requisitos
- Python 3.10+

## Ejecución

Modo consola:
```bash
python3 main.py
